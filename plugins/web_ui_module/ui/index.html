RAG Filter ⓘ 0.40

RAG Document Filter

Controls how many memory documents are included in the model's context.

  • ← Low (0.20–0.35): Includes more documents. The model has more info to answer, but may include less relevant ones.
  • → High (0.50–0.70): Strict filter. Discards most documents. The model has no context and may hallucinate.
  • Recommended: 0.35–0.45 — the model gets enough context without noise.

# Nexe Server

Local, sovereign AI server. Runs fully on your machine with configurable modules and memory.

## Privacy First
- Runs offline by default (no cloud dependency).
- Qdrant is started with `--disable-telemetry` by default.

## Quick Start
1. Run the interactive installer:
   - `./setup.sh`
   - or `python3 install_nexe.py`
2. After install, use the generated `./nexe` command.
